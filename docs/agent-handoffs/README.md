# Agent Handoff Notes

Each agent must write their handoff note here after completing their work.

## Required format:
```markdown
# Agent XX — [Name] Handoff

## What Changed
- List of files created/modified

## How to Test
1. Step-by-step verification instructions

## Known Issues / Limitations
- Any known problems

## What NOT to Merge if Failing
- Critical failures that block merge
```
